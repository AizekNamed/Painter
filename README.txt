Технический план на реализацию многофункционального графического редактора.
Программа должна выполнять следующие функции:
•	Отрисовка примитивов (круг, квадрат, линия, различные многоугольники), кривая Безье, произвольные замкнутые фигуры.
•	Смена цветов и загрузка пользовательских палитр в xml или  json.
•	Сохранение состояний пользователя с помощью сериализации или другой фичи на выбор разработчика
•	Создание совместного изображения через  сети Интернет.
•	Создание интерфейса для встраивания плагинов на других языках – Python,C#.
•	Синхронизация с мобильными устройствами изображений.
